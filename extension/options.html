Настройки расширения

Укажите, как часто расширение должно проверять цены на добавленные товары. Чем короче интервал, тем быстрее вы узнаете о скидке.

Минимальный интервал — 1 минута. Оптимально ставить от 30 до 120 минут.

Параметры доступа к Ozon Seller API

Чтобы расширение обновляло цены в фоне, укажите учётные данные Client ID и API Key, созданные в кабинете продавца Ozon. Запросы выполняются напрямую к https://api-seller.ozon.ru.

Советуем создать отдельный ключ только для чтения цен и хранить его в безопасности. Ключ сохраняется локально в браузере.
